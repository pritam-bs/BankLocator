# Bank Locator
The app shows bank locations in Estonia, Latvia and Lithuania. This includes:
1. A list of all location regions with countries as section headers (in alphabetical order).
2. A list of all locations in the selected region (in alphabetical order).
3. Details of the selected location.

## Technology
- [x] MVVM with Coordinator Pattern
- [x] Combine
- [x] BackgroundTasks
- [x] UITableViewDiffableDataSource

## Requirements
- iOS 14.0+
- Xcode 12.1
